## Nova Land

### 상태

- 기획 완료
- IA 완료
- 메인 맵 설계 완료
- 시설 설계 완료
- PC 메인 시안 진행 예정

### 컨셉

- 신규 테마파크 오픈 이벤트
- 사용자는 개장 전 점검 탐험가
- 스크롤 없는 원페이지
- 브랜드 경험 중심

### 플로우

```text
인트로
→ 탐험 시작
→ 메인 맵
→ 시설 탐험
→ 스탬프 획득
→ 4개 시설 완료
→ 신규 어트랙션 공개
→ 보상 응모
```

### 시설

- Nova Coaster
- Luna Light Garden
- Spark Energy Tower
- Wonder Parade Hall
